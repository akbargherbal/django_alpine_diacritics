Guide to Arabic Text Diacritization

Learn how to use our platform effectively for adding diacritical marks to Arabic text.

Navigation Modes

1. Manual Mode

  • Press Enter to enter edit mode
  • Use Ctrl + Space for word navigation
  • Use Space for character navigation
  • Press Escape to exit edit mode

2. Auto-pilot Mode

  • Press Ctrl + A to enter auto-pilot mode
  • System automatically moves to next position after each diacritic
  • Press Escape to exit auto-pilot mode
